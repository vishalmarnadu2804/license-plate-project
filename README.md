License Plate Recognition 🚗

📌 Project Description

This project detects and reads license plate text from car images using image processing and OCR techniques.

🎯 Objective

To extract vehicle number plate text from static images.

🛠️ Tools & Technologies Used
	•	Python
	•	OpenCV
	•	Tesseract OCR

⚙️ How It Works
	1.	Input car image
	2.	Convert image to grayscale
	3.	Apply preprocessing (resize, blur, threshold)
	4.	Extract text using OCR
	
▶️ How to Run
python3 plate.py

📷 Output
	•	Displays processed image
	•	Prints detected license plate number in terminal

📌 Note

Accuracy depends on image quality and clarity of the number plate.
